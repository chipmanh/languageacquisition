I am taking the Language Acquisition and Pedagogy class at Rhodes College. The Language Acquisition and Pedagogy class focusses on the ways of teaching and learning a foreign language, theories associated with acquiring a second language, assessment techniques, technology and curriculm development in foreign language learning settings. I took this class because I am interedted in computer science education, and wanted to see the parallels between teaching a programming language and teaching a foreign language. For my portfolio I decided to learn the web programming languages HTML and CSS to see if what we covered in the class would apply to learning computer languages as well as foreign languages.
